# FastAPI Blog Project

A simple blog API built with **FastAPI**, **SQLAlchemy**, and **Pydantic**.  
This project supports user authentication, blog CRUD operations, likes/unlikes, image uploads, and JWT authentication.

## 🚀 Features

- **User Authentication** (JWT access & refresh tokens)
- **CRUD for Blogs**
- **Upload Blog Images** via multipart form
- **Like / Unlike Blogs**
- **SQLAlchemy ORM** with relational mapping
- Auto‑generated docs via **Swagger UI**

---

## 🧱 Tech Stack

| Component         | Technology  |
| ----------------- | ----------- |
| Web Framework     | FastAPI     |
| Database ORM      | SQLAlchemy  |
| Schema Validation | Pydantic    |
| Auth              | JWT (HS256) |
| Server            | Uvicorn     |

- pip
